# Mumbai Booking App

## What it does (simple English)
This is a straightforward web application designed to help users explore exciting locations and make event bookings across Mumbai. It provides a clean, user-friendly interface to browse places and secure bookings easily.

## Tech stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Data Storage:** JSON-based local storage

## Features
- User authentication (Login and Registration system)
- Dynamic viewing of various locations in Mumbai
- Seamless booking functionality
- Lightweight and fast rendering

## Challenges I faced
- Implementing the backend routing effectively without relying on a heavy database structure.
- Linking the frontend interactions with the backend Express server using vanilla JavaScript.

## What I learned
- I learned how to set up a basic Node.js and Express server.
- I gained valuable experience handling static HTML pages and client-server communication using simple API endpoints.

## How to run
1. Clone the repo: `git clone https://github.com/kab5DeR4/Mumbai-.git`
2. Open the directory: `cd Mumbai-`
3. Install the required packages: `npm install`
4. Start the server: `node server.js`
5. Open your browser and go to `http://localhost:3000` (or the port specified in your console).
